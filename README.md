# a-shell-downloader

Script de téléchargement pour a-shell sur iPhone.
Support des liens directs, HLS et M3U8.

## Lancement

```bash
sh download.sh
$ sh download.sh


EXEMPLE
DIRECT LINK EXTRACTING

→  🎬  A  →  Movie
→  🎵  B  →  Audio
→  📝  C  →  Subtitle

Tapez A, B ou C et appuyez sur Entrez

→  Votre choix : A


Prérequis
• a-shell (gratuit sur l'App Store)
• Connexion Internet

Utilisation
1. Copie une URL (lien direct, .m3u8, ou stream
HLS)
2. Ouvre a-shell
3. Tape: sh download. sh
4. Choisis ton mode :
• A → Movie (téléchargement vidéo + audio + sous-titres)
• B → Audio (extraction audio seule)
• C → Subtitle (sous-titres seulement)

Fonctionnalités

Téléchargement de liens directs (MP4,
MKV, etc.)
• Téléchargement de streams HLS / M3U8
• Choix de la qualité vidéo (1080p, 720p,
etc.)
• Choix de la piste audio (Français, English,
etc.)
• Ajout de plusieurs pistes audio dans un
même fichier
V
Téléchargement de sous-titres
• Détection automatique des formats
disponibles
